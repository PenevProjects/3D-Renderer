// Created by Kaloyan Penev.
// Rendering program using a framework which can draw only 1 pixel at a time.

// HOW TO RUN
1. Open the solution in Visual Studio, build the solution in x86 mode and run the program.
2. Select option from the menu.
3. Evaluate.
4. Close the renderer.
5. Go to 2.

// PROBLEMS when BUILDING THE SOLUTION
If there are problems building the solution, you can:
1. Make sure the project settings have the correct includes; or
2. Copy the .cpp and .h files into a new MCG Framework project and run it; or
3. Contact me! s5121176@bournemouth.ac.uk
